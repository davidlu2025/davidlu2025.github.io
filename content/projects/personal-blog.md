## 项目简介

这个网站本身就是一个项目！基于 Hugo 静态网站生成器 + PaperMod 主题构建，完全开源免费部署。

## 技术栈

- **框架**: Hugo (Extended)
- **主题**: PaperMod
- **部署**: GitHub Pages + GitHub Actions
- **内容格式**: Markdown

## 核心功能

- ✅ 技术博客文章（支持代码高亮、目录）
- ✅ 项目展示页面
- ✅ 全文搜索（本地，无需服务器）
- ✅ 标签 / 分类系统
- ✅ RSS 订阅
- ✅ 自动暗色模式
- ✅ 移动端适配
- ✅ GitHub Actions 自动部署
